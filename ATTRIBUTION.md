## OpenStreetMap Attribution Requirements

This project contains data derived from OpenStreetMap.

When you display, publish, or redistribute this data (or a derivative database), you must provide clear credit to OpenStreetMap contributors and comply with ODbL 1.0 terms.

### Minimum Required Attribution

Use this exact credit line (or equivalent wording):

`© OpenStreetMap contributors`

Include this link with the attribution:

`https://www.openstreetmap.org/copyright`

### Website / App Footer Example

`Dive site data © OpenStreetMap contributors, ODbL 1.0`

Link `OpenStreetMap contributors` to:

`https://www.openstreetmap.org/copyright`

### API / JSON Response Example

If serving an API from this dataset, include attribution in metadata, docs, or response fields, for example:

```json
{
  "attribution": "Dive site data © OpenStreetMap contributors",
  "license": "ODbL-1.0",
  "license_url": "https://opendatacommons.org/licenses/odbl/1-0/"
}
```

### Redistribution and Share-Alike

If you publicly use or redistribute a derivative database built from OpenStreetMap data:

- Keep OpenStreetMap attribution visible and reasonable for the medium.
- License the derivative database under ODbL 1.0.
- Offer access to the derivative database under ODbL terms when required.

Official guidance:

- OSM copyright + attribution: https://www.openstreetmap.org/copyright
- ODbL 1.0 legal text: https://opendatacommons.org/licenses/odbl/1-0/
- OSMF attribution guidelines: https://osmfoundation.org/wiki/Licence/Attribution_Guidelines
